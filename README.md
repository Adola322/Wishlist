# Wishlist - система управления желаниями

## Описание
Приложение для создания списка желаний. Можно добавлять пользователей, создавать желания, отмечать выполненные.

## Как запустить

1. Установите Node.js с сайта https://nodejs.org
2. Скачайте или склонируйте репозиторий
3. Откройте терминал в папке проекта
4. Выполните команду: `npm install`
5. Запустите сервер: `node server.js`
6. Откройте браузер: http://localhost:3000

## Функции
- Добавление пользователей
- Добавление желаний
- Отметка о выполнении
- Отмена выполнения
- Редактирование и удаление желаний
- Удаление пользователей

## Технологии
- Node.js
- Express
- HTML/CSS/JavaScript
